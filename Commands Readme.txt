* Before executing any command, please execute a batch file with name "runner" present inside Playwright folder. This will keep pushing the code in background at regular interval.

Commands:
* Please change the path to Cypress folder using:
	cd Cypress

* Install all dependencies in the Cypress folder path using:
	npm install

* Run the following command to open the interactive Cypress Test Runner:
	npx cypress open

* Run the Tests in the Cypress folder path:
	npx cypress run
